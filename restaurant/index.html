⚠️ 利用前の必須セットアップ

Google Places APIは 従量課金制 です。意図しない高額請求を防ぐため、利用開始前に 2点の安全設定 を必ずお願いします。

1. Quota(リクエスト上限)の設定

  1. Google Cloud Console → APIs & Services → Places API → Quotas を開く
  2. 「Requests per day」を編集 → 50 に設定して保存
  3. これで上限超過時はAPIが停止し、課金は発生しません

2. APIキーのHTTPリファラー制限

  1. APIs & Services → 認証情報 → 該当のキーを編集
  2. アプリケーションの制限 → 「HTTPリファラー(ウェブサイト)」を選択
  3. このページを開いているURL(例: https://yourname.netlify.app/*、ローカル時は file:///*)を追加

※ 同意後はこの確認は再表示されません(localStorageで記録)
