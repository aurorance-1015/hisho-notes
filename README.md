# 秘書ノート (Hisho Notes)

> 役員秘書・経営者・士業のための業務ツール集

業務で「これ毎週やってる…自動化できるはず」と思った作業をツール化して順次公開していくシリーズです。秘書業務の **効率化と脱属人化** を目的に、ブラウザだけで動く軽量ツールを揃えていきます。

## 公開中のツール

| # | ツール | 概要 | 場所 |
|---|---|---|---|
| 1 | **会食お店ピックアップ** | エリア・条件から★4.0以上(食べログ★3.5相当)の会食店を絞り込み。食べログ検索・カレンダー登録・前日リマインド文生成までワンクリック | [restaurant/](restaurant/) |

## 開発予定

| 仮称 | 概要 |
|---|---|
| 訪問先1画面ツール | 法人番号入力で会社情報・地図・周辺情報を1画面に(国税庁API、完全無料) |
| 会議3点セットURLジェネレータ | カレンダー登録 + Zoom案内 + 出席者通知の整形を一括 |
| 年賀状リスト管理 | CSV取り込み、住所正規化、印刷用宛名出力 |
| VIPプロファイル管理 | 役員・重要顧客の好み/NG/記念日/お土産履歴を構造化(脱属人化の旗艦) |

## 設計思想

- **シングルHTML / サーバ不要**:全ツールがブラウザ単体で完結。ローカル保存(localStorage)中心
- **API利用は最小限**:無料の公的API優先。BYOK(自分のキー)時は事故防御を多層化
- **業務に持ち込みやすい**:カスタマイズはコード直書き換えで完結、シングルファイルなので渡しやすい
